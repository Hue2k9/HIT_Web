8. Write about IT security and safety rules at workplace

My office belongs to a large company, so computer security is very important. And here are some rules of information technology safety and security. First, before I leave the office, I will turn off the computer, disconnect the power, and close the door. I will create a password for my computer that only I know. I usually set a password as a string of letters and numbers, it's hard to remember, I change it often. Second, when I have an attachment in an email, I will only allow the person to whom I want to send the file to open, I will not open the file from suspicious accounts, even if someone else receives the email for security measures I use a firewall. I think this is the best way to secure my computer. Firewalls are hard to get through and my computer is always safe.
